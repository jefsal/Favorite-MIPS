# Recursion-Factorial

## Project Goal
This program computes factorial recursively to practice stack frames, saved registers, and return values in MIPS32.

## How it works
 - Sets up a stack frame for $ra and $s0 on each call
 - Uses a clear base case and returns 1 when n <= 1

### How to run
 1. Download MARS (MIPS Assembly and Runtime Simulator) IDE from the official source
    - More here dpetersanderson dot github dot io
 2. Open Recursion-Factorial.asm in MARS
 3. Assemble and Run using F3 then F11
